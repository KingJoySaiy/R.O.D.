    2.  **组合数打表**
        利用递推式`c[n][i]=c[n-1][i]+c[n-1][i-1]`打表：
```c++
const int maxn=1001;
const int mod=100;
int c[maxn][maxn];
void ini(){

    for(int n=0;n<maxn;n++){
        c[n][0]=c[n][n]=1;
        for(int i=1;i<n;i++)
            c[n][i]=(c[n-1][i]+c[n-1][i-1])%mod;
    }
}
```
